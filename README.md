# react-datetime
A date time picker base on React

## Instalation
```js
npm i @ferrwan/react-datetime
```

## Usage
#### Simple Usage
```js
...
import ReactDateTime from '@ferrwan/react-datetime'

class App extends Component {
    ...
    render() {
        return (
            <div>
                <ReactDateTime />
            </div>
        )
    }
}

```
